Detailed Description

Class representing the game board for a chess game.

The chessboard is represented as a single array of 64 integers (0 to 63). Each square on the chessboard is mapped to an integer: a1 is 0, a2 is 8, ..., h8 is 63. The mapping is done row by row from bottom to top, left to right.
